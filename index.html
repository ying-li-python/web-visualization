YT Trends

Welcome! The purpose of this project is analyze and compare Youtube content between two BuzzFeed shows, Unsolved and Taste Test.

This is important because we want to know which of these series reaches audiences in a positive way and helps us understand the type of content that is worth investing.

To answer these questions, data were retrieved from Google's Youtube API. We pulled a total of 100 videos, with 50 for each show. After assembling both datasets, we used Matplotlib to plot various video metrics vs. number of likes and used Scipy to determine linear regression and statistical significance. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations observed.
